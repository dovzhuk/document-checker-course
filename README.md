# Document Checker Course Project

Прототип системы автоматизированной проверки документов для курсового проекта.

## Идея проекта

Система принимает изображение документа или PDF, извлекает текст с помощью OCR, обрабатывает его и применяет модель машинного обучения для получения итогового результата.

## Цель

Разработать демонстрируемый CPU-only прототип, который можно показать на защите:
- загрузка документа;
- OCR;
- предобработка текста;
- предсказание модели;
- вывод результата через простой интерфейс.

## Структура проекта

- `data/` — данные проекта
- `notebooks/` — Jupyter Notebook для EDA, OCR и ML
- `src/` — основной код
- `frontend/` — интерфейс
- `artifacts/` — сохранённые модели, метрики, графики
- `demo/` — примеры для демонстрации
- `slides/` — материалы для презентации

## План реализации

1. Подготовка структуры проекта
2. Анализ данных
3. Сравнение OCR-подходов
4. Обучение ML-модели
5. Создание backend/frontend
6. Подготовка демонстрации и презентации
